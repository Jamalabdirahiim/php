## Project Overview
**TechPulse** is a modern, dark-mode technology news and review platform. It fulfills all BIT29 coursework requirements while presenting a premium, commercial aesthetic.

### Features
1.  **Tech-First Design**: Dark mode UI, neon accents, and magazine-style layout.
2.  **Role-Based Access**:
    -   **Subscribers**: Can read content and manage their profile.
    -   **Editors (Admins)**: Can manage users and publish news content to the homepage.
    -   *Auto-Role*: First registered user becomes the Editor.
3.  **Dynamic Content**: Homepage sections (Hero, About, Footer) are managed via the CMS Dashboard.
4.  **Security**: Password hashing, session timeouts, and prepared statements.

### Installation
1.  **Database**:

    -   Import `database.sql` found in this folder.
